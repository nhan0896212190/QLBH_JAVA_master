# QLBH_JAVA
Hướng dẫn cài đặt
1. Tải jdk 8
2. Tải netbeans 8
3. Download plugins -(https://sourceforge.net/projects/ireport/files/iReport%20Plugin%20for%20NetBeans/iReport-5.6.0/iReport-5.6.0-plugin.zip/download)
4. add plugin (vào netbeans -> Tools -> Plugins -> Downloaded -> Add plugins... -> Chọn các file .nbm trong file zip vừa tải
5. add các file jar
6. chạy SQL

8. Mật khẩu : admin1
